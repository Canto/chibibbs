** 라이센스 **
1. 본 게시판은 오픈소스 라이센스 규약인 GPLv3 에 의거하여 배포됩니다.
2. 게시판의 수정과 재배포에 있어서 규제를 하지않습니다.
   [ 다만 수정 재배포를 하시기 전에 Canto(http://canto.btool.kr) 혹은 
     커뮤모아(http://commumoa.net)에 연락을 주시면 감사하겠습니다. ]
3. 스킨 제작자 분들은 따로 연락을 주시면 사용된 변수/함수등 스킨제작에 있어
   최대한 서포트 해 드리겠습니다.
4. 게시판 및 ChibiPAINT프로그램에 관련된 오류나 버그는 제작자 개인홈이나
   배포페이지를 이용해주세요.
5. 본 게시판을 이용하기 위해서는 JAVA 가 필요합니다.
   JAVA를 설치하지 않은 분은 
   http://www.java.com/ko/download/ie_manual.jsp?locale=ko&host=www.java.com
   에서 다운로드를 해 주시기 바랍니다.
