4. 게시판 및 ChibiPAINT프로그램에 관련된 오류나 버그는 제작자 개인홈이나
   배포페이지를 이용해주세요.
5. 본 게시판을 이용하기 위해서는 JAVA 가 필요합니다.
   JAVA를 설치하지 않은 분은 
   http://www.java.com/ko/download/ie_manual.jsp?locale=ko&host=www.java.com
   에서 다운로드를 해 주시기 바랍니다.
